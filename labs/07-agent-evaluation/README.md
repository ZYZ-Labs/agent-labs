# Lab 07: Agent Evaluation

## Objectives

- Evaluate agent outputs against a reference answer with an LLM-as-judge.
- Add deterministic rule-based checks.
- Collect pass/fail metrics across a small test set.

## Run

```bash
cd python
pip install -r requirements.txt
python main.py
```

## Expected Output

A report showing which test cases passed rule checks and LLM judge scores.
